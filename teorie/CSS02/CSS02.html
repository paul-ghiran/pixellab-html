Proprietati Modificare Text

Font-weight

font-weight defineste grosimea textului

Are valori numerice ( 100, 200, 300, 400, 500, 600, 700, 800, 900)

100 inseamna cel mai subtire text, 900 cel mai ingrosat

Mai poate primi valori precum "light", "normal" sau "bold" - in functie de font.

Font Size

Altereaza marimea textului.

Posibile unitati de masura:

  • Px - Unitate de masura in pixeli
  • Em - Unitate de masura relativa la parinte (multipli de 1, unde 1=marimea parintelui)
  • Rem - Unitate de masura relativa la root element(root element = html) unde 1= aceeasi marime cu root

Unitati de masura mai putin folosite:

  • % - Procent din font size-ul parintelui, 100% = 1em
  • vw - Procent din latimea browserului, 100vw = un browser intreg
  • vh - Procent din inaltimea browserului, 100vw = un browser intreg

Line Height

Line-height ne spune inaltimea unui rand de text (unei linii) incluzand spatiul. Un line-height mai mare nu va face un font-size mai mare.

Se exprima in aceleasi unitati precum mai sus. Bineinteles, unitatile se vor putea folosi si la alte proprietati pe viitor.

Text Decoration

Decoreaza textul. In general folosim underline overline, line-through si none pt a scoate tecoration-ul.

Font Family

font-family reprezinta cum arata caracterele.

Familii de fonturi:
  • Serif
  • Sans-serif

Un text poate avea o insiruire de font-families in functie de care sunt disponibile sau nu pe device, incarcate sau nu. Aceste font-families sunt specificate cu apostroafe si sunt separate prin virgula.

Custom Fonts

Fonturi create de designeri sau cumparate online, acestea trebuie prima data incarcate in website, pentru ca nu putem sa ne bazam ca utilizatorul le va avea instalate pe device.

Pentru a le folosi, recurgem la regula @font-face

Pentru fonturi vom folosi in general Google Fonts.

Text-Shadow

Afiseaza o umbra sub un text, i se da un offset (spatiu) orizontal, unul vertical, marimea umbrei(cat este de intinsa) si o culoare dorita.

Ex: text-shadow: 2px 2px 3px red;

Spacing & shaping

Box Model

Avem 2 tipuri principale de spatiere. Margin- Spatierea din afara elementului, Padding: Spatierea din interiorul elementului.

Acestea pot fi vazute in dev tools, margin portocaliu, padding: verde

Padding si margin pot fi exprimate in px, vw, vh, procent (care se raporteaza la latimea parintelui) cat si em si rem.

Acestea pot fi declarate separat, explicit, precum pe elementul acesta (vezi CSS)

La fel si la margin, spatierile per directie pot fi declarate separat, explicit, precum pe elementul acesta (vezi CSS)

Putem atat la padding cat si la margin fie o valoare, 2 sau 4, care vor fi un shorthand notation in felul urmator:
1: inseamna peste tot
2: inseamna prima valoare sus-jos, a doua stanga-dreapta
4: inseamna pe rand, sus dreapta jos stanga (mai greu de retinut poate)

La Margin putem da valoarea orizontala "auto" atunci cand un obiect are width, si acesta se va centra.

Width & Height

Proprietatile acestea specifica dimensiunile unei "cutii" sau ale unui element. Pot fi folosite unitatile de mai sus, px, %, vw, vh, em, rem.
